# SmarkStock Desktop — install guide

## What this is

A small Windows app that sources BOM parts automatically using an AI browser
agent. Version 0.1.0.

## 1. Download

Get the installer here: **[Download SmarkStock Desktop](https://drive.google.com/file/d/1h2WEj31eDA_DsyHyQig_EL2Ktau-LSrL/view?usp=sharing)**

File: `SmarkStock Desktop_0.1.0_x64-setup.exe` (~29.5 MB)

## 2. Install

1. Double-click the downloaded `.exe`.
2. If Windows shows a "Windows protected your PC" screen, click **More info**
   → **Run anyway** (this is normal for a new, unsigned internal app).
3. Follow the setup wizard — Next, Next, Install, Finish.
4. Launch **SmarkStock Desktop** from the Start menu.

## 3. One-time setup (before your first sourcing run)

Two things you need on your own PC — these can't be installed automatically:

1. **Claude Code CLI** — install it, then open any terminal (PowerShell/Command
   Prompt) and type:
   ```
   claude
   ```
   Sign in with your own Claude account/subscription when prompted. You only
   need to do this once per PC.

2. **Brave browser** — download and install from [brave.com](https://brave.com).
   (Google Chrome also works if you already have it, but Brave is recommended.)

That's it — no other setup needed. Full details are also inside the app
itself: click **Setup & help** on the BOM list screen any time.

## 4. Using the app

1. Log in with your normal SmarkStock username and password.
2. Pick a BOM from the list.
3. Leave the line limit blank (sources everything) or set a small number to
   test first.
4. Click **Start sourcing** and leave it running — it works on its own, no
   need to watch it.
5. When it finishes, click **Open review on the web** to check the results.
